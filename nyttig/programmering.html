Tilfeldige tall, avrunding, sammenlikningsoperatorer og if-setninger

Variabler brukes til å holde orden på verdier underveis i et program. Slike verdier kan være tekst (f.eks. navn), tall (f.eks. konstanter) eller de boolske verdiene sann eller usann.
Dersom vil ønsker å finne ut hva slags variabel vi har å jobbe med kan vi logge
   console.log(typeof vairabelnavn); F.eks. vil resultatet av følgende kode være
   var tallVar = 1;
   log (typeof tallVar); // returnerer number
   var tekstVar = "tekst";
   console.log(typeof tekstVar); // returnerer string
   var testVar = "";
   console.log(typeof testVar); // returnerer string
   var tallTekst = "1";
   console.log(typeof tallTekst); // returnerer string
   var tomVar
   console.log(tomVar); // returnerer undefined
   var boolskVar = true;
   console.log(typeof boolskVar); // returnerer boolean

Vi kan gjøre om en numbervariabel til en tekstvariabel med
   var tallTekstVar = 3489;
   console.log(typeof tallTekstVar); // returnerer number
   var tekstFraTall = String(tallTekstVar);
   console.log(typeof tekstFraTall); // returnerer string
Dette kan vi få bruk for når vi for eksempel skal finne antall siffer i et tall. Antall siffer vil være lengden av stringen:
   console.log(tekstFraTall.length); // returnerer 4

Tallvariabler kan opprettes ved å tilordne variabelen en kjent verdi, f.eks.
   var tall = 7; eller vi kan tilordne variabelen en tilfeldig verdi.

Metoden Math.random(); genererer et tilfeldig tall mellom 0 og 0,99999999999999999.... Dersom vi ønsker å ha et heltall, må vi

  1. Multiplisere det med den (øvre grensen + 1) for hvilket tall vi ønsker å ha
    • Ved å gange det tilfeldige tallet med 11 får vi et tilfeldig desimaltall mellom 0 og 10
  2. Runde tallet ned til nærmeste heltall for å slippe desimaltall. Dette gjøres med metoden floor (rund ned)
    • Ved å runde ned får vi et tilfeldig heltall mellom 0 og 10

Altså:
   var tilfeldig = Math.random();
   console.log(tilfeldig); // returnerer et tilfeldig desimaltall mellom 0 og 0,99999
   tilfeldig = tilfeldig * 11;
   console.log(tilfeldig); // returnerer et tilfeldig desimaltall mellom 0 og 10,9999999
   tilfeldig = Math.floor(tilfeldig);
   console.log(tilfeldig); // returnerer et tilfeldig heltall mellom 0 og 10
Dette kan også skrives enklere:
   var tilfeldigTall = Math.floor(Math.random()*11);
   console.log(tilfeldigTall);

Det finnes også andre avrundingsmetoder. Avrundingen Math.ceil(tall) runder tallet opp til nærmeste heltall og avrundingen Math.round() runder av tallet slik vi gjør i matematikken. Dersom du ikke ønsker nærmeste heltall kan du bestemme antall desimaler i tallet med tall.toFixed(antall):
   var tilfeldig = Math.random();
   console.log(tilfeldig); // returnerer f.eks. 0.8656078085086074
   var tilfeldigOpp = Math.ceil(tilfeldig*10);
   console.log(tilfeldigOpp); // returnerer da 9
   var tilfeldigNed = Math.floor(tilfeldig*11);
   console.log(tilfeldigNed); // returnerer da 9
   var tilfeldigRound10 = Math.round(tilfeldig*10);
   console.log(tilfeldigRound10); // returnerer da 9
   var tilfeldigRound11 = Math.round(tilfeldig*11);
   console.log(tilfeldigRound11); returnerer da 10
   var tilfeldigToDesimaler = (tilfeldig*10).toFixed(2); // runder av matematisk til antall siffer
   console.log(tilfeldigToDesimaler); // returnerer da 8.66
   console.log(typeof tilfeldigToDesimaler); // OBS! returnerer string

Hva bruker vi tilfeldige tall til?

I en tekstvariabel kan vi plukke ut et tilfeldig tegn ved å opprette et tilfeldig tall mellom 0 og (lengden av teksten)-1. I en array kan vi hente ut en tilfeldig verdi ved å opprette et tilfeldig tall mellom 0 og (lengden av arrayen)-1. På denne måten kan vi f.eks. ha tilfeldige løsningsord i en hangman, vi kan lage en quiz der svaralternativene kommer i tilfeldig rekkefølge osv. Ved å lage tre tall mellom 0 og 255 kan vi lage en funksjon som returnerer tilfeldige farger etter rgb-modellen.

Forløkker

Forløkker er fine å bruke når vi vet på forhånd hvor mange ganger vi skal løpe gjennom en kode. Et eksempel på dette er når vi skal utføre en operasjon for en hel datakolleksjon. Et annet eksempel er når vi skal gjøre en operasjon et visst antall ganger, som i dette eksempelet der vi lager en tabell med den store gangetabellen:
   document.write('< table >');
   for (var i=1; i<=10; i++){
     document.write('< tr >');
     for(var j=1; j<=10; j++){
       document.write('< td >'+i*j+'< /td >');
     }
     document.write('< /tr >');
   }
   document.write('< /table >');

En forløkke kan også telle nedover:
Skriv kode for å telle ned partallene fra 50 til 30
   for(i = 50; i>= 30; i-=2){ // i = i-2
     //if(i % 2==0){ // if-setningen er ikke nødvendig når vi teller annenhver
     console.log(i);
     //}
   }

Eksempel med forløkker og datakolleksjoner kommer når vi repeterer datakolleksjoner.

Whileløkker

I en whileløkkene har vi ikke en teller som gir antall repetisjoner av koden, men en betingelse som sier om koden skal kjøres. Den kan derfor sammenlignes med et ukjent antall if-else-setninger. Et eksempel der vi trenger en whileløkke er når vi skal finne ut hvor mange ganger vi må "kaste to terninger" før vi får to seksere:
   var terning1 = 0;
   var terning2 = 0;
   var teller = 0;
   while (terning1!==6 || terning2!==6){
     terning1 = Math.floor(Math.random() * 6) + 1;
     terning2 = Math.floor(Math.random() * 6) + 1;
     teller++
   }
console.log("det tok " + teller + " forsøk før terningene viste " + terning1 +" og " + terning2);


Typiske feil man gjør når man bruker whileløkker er at betingelsen aldri blir sann slik at løkken kjører uendelig antall ganger (hvis man ikke endrer verdi på begge terningene i eksempelet over).
