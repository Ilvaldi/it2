Datakolleksjoner

Vi repeterer datakolleksjonene array og object samt hvordan vi kan kombinere disse. Dette er et viktig grunnlag før vi går videre og kombinerer det med brukerinput!

Array

Vi definerer en array på samme måte som en variabel:
   var tallArray
Hvis vi vil opprette en array uten innhold sånn at vi har bestemt datatypen skriver vi
   var tallArray = []
Når vi fyller arrayen med verdier samler vi alt som skal lagres i arrayen mellom [] og skiller verdiene med komma:
   var tallArray = [0, 1, 2, 3, 4, 5];
Variablen over er array med lengde 6. En array er indeksert fra 0 til lengden-1 der arrayens lengde er antall verdier
Vi henter ut enkeltverdier med
   console.log(tallArray[0]) // returnerer 0
og finner lengden med
   console.log(tallArray.length) // returnerer 5

Vi endrer verdier i en array ved:
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray); // returnerer [0, 1, 2, 3, 4, 5]
   tallArray[2] = 42;
   console.log(tallArray); // returnerer [0, 1, 42, 3, 4, 5]

Vi legger til verdier i en array ved metodene arrayNavn.push(verdi) (legger til på slutten av arrayen) og arrayNavn.unshift(verdi) (legger til på starten av arrayen):
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray); // returnerer [0, 1, 2, 3, 4, 5]
   tallArray.push(42);
   console.log(tallArray); //returnerer [0, 1, 2, 3, 4, 5, 42]
   tallArray.unshift(42);
   console.log(tallArray); // returnerer [42, 0, 1, 2, 3, 4, 5, 42]

Vi fjerner verdier i en array ved metodene arrayNavn.pop() (fjerner på slutten av arrayen) og arrayNavn.shift() (fjerner på starten av arrayen):
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray); // returnerer [0, 1, 2, 3, 4, 5]
   tallArray.pop();
   console.log(tallArray);// returnerer [0, 1, 2, 3, 4]
   tallArray.shift();
   console.log(tallArray); // returnerer [1, 2, 3, 4]

Vi finner enkeltverdier i en arrray ved metoden arrayNavn.indexOf(verdi) som gir oss plasseringen til den første forekomsten av en gitt verdi (senere skal vi klare å finne alle):
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray); // returnerer [0, 1, 2, 3, 4, 5]
   console.log(tallArray.indexOf(2)); // returnerer 2
   tallArray.unshift(2);
   console.log(tallArray); // returnerer [2, 0, 1, 2, 3, 4, 5]
   console.log(tallArray.indexOf(2)); // returnerer 0

Vi fjerner enkeltverdier midt i arrayen ved metoden arrayNavn.splice(index, antall verdier). Her kan indexen finnes i samme operasjon: arrayNavn.splice(arrayNavn.indexOf(verdi), antall verdier):
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray); // returnerer [0, 1, 2, 3, 4, 5]
   tallArray.splice(tallArray.indexOf(1), 1);
   console.log(tallArray); // returnerer [0, 2, 3, 4, 5]

Vi kan også legge til enkeltverdier midt i arrayen ved bruk av arrayNavn.splice(indexForForsteNye, antallSlettes, verdiSomLeggesTil, verdiSomLeggesTil osv.)
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray); // returnerer [0, 1, 2, 3, 4, 5]
   tallArray.splice(3, 0, 42, 42, 42);
   console.log(tallArray); // returnerer [0, 1, 2, 42, 42, 42, 3, 4, 5]
   tallArray.splice(3, 2, 11, 12, 'tall? tull?');
   console.log(tallArray); // returnerer [0, 1, 2, 11, 12, "tall? tull?", 42, 3, 4, 5]

Vi bruker forløkker til å løkke gjennom arrays. Dersom vi ønsker at samme kode skal gjennomføres for alle elementer i en array setter vi startverdi for telleren til å være i = 0 og lar løkken løpe til i < arrayens lengde:
   for (var i = 0; i < arrayen.length; i++){
     // kode
   }

Vi kan for eksempel bruke dette til å skrive ut alle elementene i en array (ikke så spennende):
   var tallArray = [1, 3, 5, 8, 10, 2, 44, 0];
   for(var i = 0; i < tallArray.length; i++){
     console.log(tallArray[i]);
   }

Metoden querySelectorAll

Vi husker fra tidligere at metoden querySelector finner første forekomst av et element (querySelector("p")), av et element med en gitt klasse (querySelector(".klassenavn")) eller av et element med en gitt id (querySelector("#idnavn")). Det ble også nevnt at det finnes en metode querySelectorAll som tar inn alle elementer av en gitt type/klasse. Denne har vi spart fordi disse variablene lagres som nettopp en array. For å bearbeide dem må jeg da hente inn alle ved hjelp av en løkke.

Som eksempel kan jeg hente inn alle td-elementer i dette dokumentet og gi dem rosa bakgrunn:
   var tdEl = document.querySelectorAll("td");
   for(var i = 0; i < tdEl.length; i++){
     tdEl[i].style.backgroundColor = "pink";
   }

Object

I en array er det plassen til hver verdi vi viser til når vi skal bruke den. Objects kan sammelgnes med det vi i php kalte assosiative arrays, der hver verdi har en nøkkel vi refererer til når vi skal ha verdien. Vi lagrer hver enkelt verdi sammen med en nøkkel som sier noe om verdien (tilsvarer assosiative arrays i php):
   var objectVar = {nøkkel1: "verdi1", nøkkel2: "verdi2", osv.}
Som et eksempel kan jeg lagre personinfo i et object:
   var person = {fnavn: "Asta", enavn: "Fjærli", alder: 28, hobby: "løfte tungt"};
Mer oversiktlig er det å skrive
   var person = {
     fnavn: "Asta",
     enavn: "Fjærli",
     alder: 28,
     hobby: "løfte tungt"
   };
Som vi ser, kan objectverdiene inneholde både tekst og tall. De kan også inneholde boolske variabler og arrays.

I en array finner man verdien ved å referere til plasseringen og bruke klammenotasjon: array[plass]. I en object refererer vi til nøkkelen og bruker enten klammenotasjon eller dotnotasjon:
   console.log(person["fnavn"], person.fnavn);
   console.log(person["hobby"], person.hobby);

Av og til tilhører det flere verdier til hver nøkkel, f.eks. kan man ha flere hobbyer. Da legger man disse i en array til nøkkelen, dette er nyttig for å unngå et uvisst antall nøkler hobby1, hobby2, osv, og det gir oss muligheten til å bruke de vanlige metodene på arrayen:
   var person = {
     fnavn: "Asta",
     enavn: "Fjærli",
     alder: 28,
     hobby: ["spille fløyte", "løfte tungt", "strikke"]
   };

For å hente ut en gitt hobby holder det ikke å be om person["hobby"], da det vil returnere
   (3) ["spille fløyte", "løfte tungt", "strikke"]
For å få ut en viss hobby må man derfor be om person["hobby"][1];

Endre verdier i objects

På samme måte som i arrays kan vi overskrive enkeltverdier i objects. Her kan vi bruke både klammenotasjon og dotnotasjon:
   person.fnavn = "Asta Rønning";
   person["alder"] = 1000;
   person.hobby[0] = "korps";
   person["hobby"][1] = "vektløfting";

På samme måte kan vi legge til verdier:
   person.favorittbok = "Ringdrotten";
   person["favorittland"] = "Island";
   person["hobby"].push("turn");

Og vi kan slette verdier:
   delete person.enavn;
   delete person["fnavn"];

Løkke gjennom objects

Vi kjenner til forløkker som for(var i = startverdi; i <= sluttverdi; i++) der vi vet antall ganger koden skal gjentas. For objects kjenner vi ikke nødvendigvis til antall egenskaper, og kan heller ikke referere til objectets lengde. Forløkker for objects kaller vi derfor for - in - løkker. Her konstrueres en egenskapsvariabel som sørger for at alle egenskaper i et object blir løkket gjennom. Denne egenskapsvariablen kan vi kalle hva som helst: kodene
   for(var egenskap in person){
     console.log(egenskap);
   }
og
   for(var jul in person){
     console.log(jul);
   }
returnerer begge alle egenskaper i objectet. Her må vi bruke klammenotasjon for å få ut verdiene, vi kan ikke bruke dot-notasjon.
Vi kan skrive ut enten nøkkelen eller verdien:
   for(var egenskap in person){
     console.log(egenskap, person[egenskap]);
   }
   for(var jul in person){
     console.log(jul, person[jul]);
   }

This-notasjon

Et object kan, i tillegg til egenskaper, ha metoder. Disse fungerer som funksjoner, og oppgaven til en slik metode kan for eksempel være å returnere alder og favorittland, eller å skrive ut en liste over hobbyene. Fordi funksjonen er definert inne i objectet, bruker vi kodeordet this for å peke på egenskapen. Det forteller at vi skal finne egenskapen i nettopp dette objectet.

   var person = {
     fnavn: "Asta Rønning",
     enavn: "Fjærli",
     alder: 28,
     hobby: ["korps", "vektløfting", "strikke", "turn"],
     favorittbok: "Ringdrotten",
     favorittland: "Island",
   // returnere alder og favorittland
     alderLand: function(){
       return this.alder + " " + this.favorittland;
     },
   // skrive ut en liste over hobbyer
     hobbyer: function(){
       var bodyEl = document.querySelector("body");
       var liste = document.createElement("ul");
       for(i = 0; i < this.hobby.length; i++){
         var listeEl = document.createElement("li")
         listeEl.innerHTML = this.hobby[i];
         liste.appendChild(listeEl)
       }
       bodyEl.appendChild(liste);
     }
   }
   console.log(person.alderLand());
   person.hobbyer();

Kombinasjoner av arrays og objects

Arrays med arrays

Todimensjonale arrays er arrays med arrays inne i:
   var A = [
     [1, 2, 3],
     [4, 5, 6]
   ]

For å løkke gjennom disse må vi bruke to for-løkker:
   for(var i = 0; i < A.length; i++){
     for (var j = 0; j < A[i].length; j++){
       console.log(A[i][j]);
     }
   }

Objects med arrays

Så langt har vi laget objects med arrays i stedet for enkeltverdier:
   var person = {
     fnavn: "Asta",
     enavn: "Fjærli",
     alder: 28,
     hobby: ["spille fløyte", "løfte tungt", "strikke"]
   };

Dette er nyttig for å unngå et uvisst antall nøkler hobby1, hobby2, osv, og det gir oss muligheten til å bruke de vanlige metodene på arrayen:
   person.hobby[1] = "vektløfting";
endrer en verdi i arrayen i objectet ved å referere til plasseringen i arrayen.
   person.hobby.push("turn");
legger til en verdi sist i arrrayen i objectet ved metoden push.
   person.hobby.pop();
fjerner den siste verdien i arrayen ved metoden pop. Vi kan også bruke shift, unshift og splice til å legge til og fjerne verdier

Arrays med objects

Vi kan også få bruk for å lagre en array som inneholder objects i stedet for enkeltverdier. Som eksempel bruker vi elever i en klasse. Disse elevene får hver sin plass i arrayen:
   var elever = ["Andrea", "Damir", "Silje", "Thomas"];
Dersom vi ønsker å lagre mer informasjon enn bare navnet på disse elevene i arrayen, får vi et problem. Det gir jo ikke mening å sette inn Andrea s etternavn som elever[4]. Da er det bedre å legge inn et object for hver elev og la denne objecten ha egenskaper som fornavn, etternavn, alder, telefonnummer osv. Da kan vi godt lage elevenes objects først, og deretter legge dem inn i en array etterpå.
   var elev1 = {
     fnavn: "Andrea",
     enavn: "Fredriksen",
     alder: 18
   };
   var elev2 = {
     fnavn: "Damir",
     enavn: "Hansen",
     alder: 19
   };
   var elev3 = {
     fnavn: "Silje",
     enavn: "Aasen",
     alder: 17
   };
   var elev4 = {
     fnavn: "Thomas",
     enavn: "Fris",
     alder: 18
   };
   var elever = [elev1, elev2, elev3, elev4];
Eller vi kan legge inn objectene direkte i arrayen:
   elever.push({fnavn: "Silje", enavn: "Evensen", alder: 19});

Om vi lager objectene først som egne variabler og deretter legger dem inn eller om vi legger hele objects i arrayen kommer helt an på hva vi synes er mest praktisk og om vi starter fra ingenting eller har en eksisterende array/eksisterende objects fra før.

Vi henter ut hele objectet ved å referere til plassen: elever[1] gir ut hele objectet til Damir. Skal vi ha ut kun en egenskap, refererer vi til nøkkelen ved enten dot-notasjon eller klammenotasjon: elever[1].fnavn og elever[1]["fnavn"] gir begge ut fornavnet.

Å løkke gjennom arrays med objects

Siden arrays med objects fortsatt har en gitt lengde, dvs en gitt mengde objects, bruker vi en vanlig forløkke for å bruke alle objectene. Vi kan for eksempel finne gjennomsnittsalderen:
   var alderSum = 0;
   for(var i = 0; i < elever.length; i++){
     alderSum += elever[i].alder;
   }
   var snittAlder = alderSum/elever.length;
   console.log(snittAlder.toFixed(2));

Å sortere objects i en array

For arrays har vi den innebygde sorteringsmetoden sort(arrayNavn); som sorterer en tekstarray i alfabetisk rekkefølge.
   var tekst = ["def", "aeg", "abc", "aaaa"];
   tekst.sort();
   console.log(tekst);
Denne må utvides for å også gjelde tall: koden under fungerer ikke optimalt og returnerer [-1, -20, 0, 19, 2, 4] fordi 2 er større enn 1, og maskinen derfor tror at -20 er større enn -1 (håpløst).
   var tall = [2, 0, -1, 4, 19, -20];
   tall.sort();
   console.log(tall);
Her må vi fortelle at sorteringen skal fortsette til arrayen faktisk er sortert. Dette implementerer vi ved å legge inn en funksjon i sorteringen som finner forskjellen mellom tallene og sorterer ved å sjekke om forskjellen mellom to tall er positiv +1 (den andre verdien er mindre enn den første), null 0 (verdiene er like store) eller negativ -1 (den andre verdien er større enn den første):
   var tall = [2, 0, -1, 4, 19, -20];
   tall.sort(function(a, b){return a-b});
   console.log(tall);

Vi kan skrive ut hele denne sorteringen for å kunne utvide den til å sortere en array med objects etter en av nøklene (her må vi prioritere hvilken nøkkel vi skal sortere etter):
   var tall = [2, 0, -1, 4, 19, -20];
   tall.sort(function(a, b){
     if(a > b){
       return 1;
     }
     else if(a < b){
       return -1;
     }
     else{
       return 0;
     }
   });
   console.log(tall);

Skal vi sortere en array med objects bruker vi samme konseptet, men spesifiserer hvilken nøkkel vi sorterer på:
   elever.sort(function(a, b){
     if(a.enavn > b.enavn){
       return 1;
     }
     else if(a.enavn < b.enavn){
       return -1;
     }
     else{
       return 0;
     }
   })
   console.log(elever);

Å skrive ut objects i en array til en tabell på nettsiden

To av bruksområdene til arrays med objects er å oversiktlig kunne skrive ut data til siden, for eksempel i tabellform, og å kunne lagre inputverdier fra skjemaer oversiktlig. Når vi skal skrive ut data til en tabell, bruker vi nøklene som kolonneoverskrifter og fyller inn hver arrayverdi i en ny rad. I dagens eksempel vil det være å skrive ut en tabell med kolonnene fnavn, enavn og alder der hver elev får sin rad. Vi starter helt fra scratch med å lage selve tabellelementet:

   var bodyEl = document.querySelector("body")
   var tabellEl = document.createElement("table");
   var overskriftEl = document.createElement("tr");
   overskriftEl.innerHTML="< th >Fornavn< /th >< th >Etternavn< /th >< th >Alder< /th >";
   tabellEl.appendChild(overskriftEl);
   for(var i=0; i < elever.length; i++){
     var radEl = document.createElement("tr");
     radEl.innerHTML = "< td >"+elever[i].fnavn+"< /td >"+"< td >"+elever[i].enavn+"< /td >"+"< td >"+elever[i].alder+"< /td >";
     tabellEl.appendChild(radEl);
   }
   bodyEl.appendChild(tabellEl);

Deretter kan vi stilsette tabellen som vi ønsker.
