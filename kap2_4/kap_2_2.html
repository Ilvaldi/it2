De tre datatypene

String

Datatypen string brukes for tekstvariabler. All tekst skal legges i fnutter ('tekst') eller dobbeltfnutter ("tekst"):
   < script >
     var tekstVar = 'En setning med tekst!';
  < /script >

En tekststreng er indexert fra 0 til (n-1) der n er antall tegn. Vi kan behandle tekstvariabler ved flere metoder, blant annet

Metode Forklaring Eksempel
[plass] Gir tegnet på en gitt plassering console.log(tekstVar[3]);
returnerer s
indexOf() Finner plasseringen til første element av typen et spørres etter console.log(tekstVar.indexOf('n'));
returnerer 1
toUpperCase() Gjør om til kun store bokstaver console.log(tekstVar.toUpperCase());
returnerer EN SETNING MED TEKST!
toLowerCase() Gjør om til kun små bokstaver console.log(tekstVar.toLowerCase());
returnerer en setning med tekst!
length Gir antall tegn i stringen console.log(tekstVar.length);
returnerer 21
substring() Gir tegnene i et intervall console.log(tekstVar.substring(11, 14));
returnerer med

Boolean

Vi har to boolske variabler: sann (1) og usann (0). Disse legger blant annet grunnlag for if-setninger, der vi ber om at kode skal kjøres dersom en påstand er sann eller usann.
Boolske variabler lagres som
   < script >
     var boolskVar = true;
     var boolskVar = false;
     var boolskVar = 1;
     var boolskVar = 0;
   < /script >

Number

Number er alle tallvariabler: heltall, desimaltall, negative tall, positive tall. Merk at JavaScript ikke skiller mellom heltall og desimaltall!
Vi lagrer numbervariabler som
   < script >
     var tallVar = 1;
     var tallVar = 1.0;
     var tallVar = -157.42
   < /script >

Sjekke datatype

Dersom vi lurer på hva slags datatype en variabel har kan vi sjekke det i konsollen. Skriv inn følgende kode og se hva som logges:
   < script >
     var heltall = 13;
     var decimaltall = 155.34;
     var tekst = '10';
     var boolsk = true;
     var boolskTall = 0;
     console.log(typeof heltall, typeof decimaltall, typeof tekst, typeof boolsk, typeof boolskTall);
   < /script >

Ekstra: må vi alltid skrive var?

JavaScript er et av språkene som godkjenner mye syntax som egentlig ikke er lovlig. Hva skjer hvis du skriver inn følgende kode?

   < script >
     fornavn = "Kaia";
     fodselsar = 1900;
     console.log(fornavn + ' ville vært ' + (2020-fodselsar) + ' år gammel.');
   < /script >

I IT-2-faget holder det å alltid bruke var for nye variabler, uansett hvilken datatype de har. Selv om det ofte går fint uten var også, skal dere bruke det av god vane.

Dersom du er vant til ulike måter å deklarere ulike variabler på kan vi bruke både var, let og const. let og const har det vi kaller Block Skope, som er å bevege seg et godt stykke ut fra hva som forventes i IT-2. Du kan lese mer om det på w3schools.
